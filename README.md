## 📊 Project Overview
This project involves building an interactive dashboard in Power BI to analyze Amazon sales data and track key KPIs such as revenue, orders, and reviews.

## ⚙️ Tools Used
- Power BI  
- DAX  
- Power Query  

## 📈 Features
- KPI tracking (Revenue, Orders, Reviews)  
- Sales trend analysis  
- Interactive filters and slicers
